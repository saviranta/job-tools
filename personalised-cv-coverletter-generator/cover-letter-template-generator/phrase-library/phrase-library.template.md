# Phrase Library

A personal collection of phrases, transitions, and lines that work for you.
The cover-letter-generator draws from this to keep language fresh and on-voice.

**How to use:**
- Add phrases as you write cover letters you're happy with
- Tag each phrase with tone: [formal] [warm] [direct] [narrative] [bold]
- The cover-letter-generator will pick phrases matching the template's tone setting
- Remove or archive phrases that start to feel stale

**How to add new phrases:**
Use the quick-add format: paste a phrase and its section, Claude tags it and adds it here.
